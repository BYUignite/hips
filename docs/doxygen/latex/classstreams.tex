\doxysubsection{Detailed Description}
Class implementing streams for use in mixing and or reaction problems. This is writting in terms of mixture fraction with streams defined in an input file. The class can implement products of complete combustion, or equilibrium (through the Cantera Ideal\+Gas\+Phase object, if desired). This class holds a pointer to a Cantera Ideal\+Gas\+Phase object (defined up front in main) that computes thermodynamic, kinetic, and transport data.
